## Завдання 2: Обчислення визначеного інтеграла

### Висновок

У другому завданні я реалізувала метод Монте-Карло для наближеного обчислення інтеграла функції `f(x) = x^2` на інтервалі від 0 до 2. Я порівняла результати з точним значенням інтеграла, яке отримала за допомогою функції `quad` з SciPy.

Я отримала такі значення:  
- Метод `Монте-Карло`: `2.716`  
- Функція `quad`: `2.666666666666667`

Метод Монте-Карло дає наближене значення інтеграла, яке залежить від кількості випадкових точок (`n`). Збільшення `n` може покращити точність. Функція `quad` дає точне значення інтеграла, яке можна використовувати для перевірки точності методу Монте-Карло.
